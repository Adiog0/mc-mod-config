Minecraft Mod Config Editor — Icon Reference
=============================================

Diretório: icons/
Formato: PNG com fundo transparente
Tamanhos padrão: 24x24 para botões/toolbar, 16x16 para tree items


═══ APP / TOOLBAR (24x24 px) ═══════════════════════════════════════════

pickaxe.png    — machado de mineracao do Minecraft (icone do app, titulo da janela)
castle.png     — castelo ou construcao (label "Voce esta editando: ...")
folder.png     — pasta aberta (botao "Abrir Instancia...")
refresh.png    — setas circulares (botao "Recarregar")
palette.png    — paleta de cores (menu "Carregar CSS Customizado")
save.png       — disquete ou baú (botoes "Salvar" e "Backup")
undo.png       — seta curvada para esquerda (botao "Cancelar")


═══ TREE ITEMS (16x16 px) ══════════════════════════════════════════════

block.png      — bloco de terra ou grama do Minecraft (mods na arvore)
settings.png   — engrenagem (arquivos TOML)
crafting.png   — bancada de craft ou martelo (arquivos JSON / JSON5)
scroll.png     — pergaminho enrolado (arquivos YAML)
file.png       — folha de papel generica (outros formatos: CFG, TXT, etc)


═══ BOTOES DE ACAO (24x24 px) ══════════════════════════════════════════

add.png        — sinal de mais verde (botao "Adicionar Parametro")
delete.png     — X vermelho (botao de excluir parametro)
wrench.png     — chave inglesa (placeholder "Selecione um arquivo...")


═══ STATUS (16x16 px) ══════════════════════════════════════════════════

check.png      — visto verde (mensagens de sucesso)
error.png      — X vermelho (mensagens de erro)


═══ NOTAS ═══════════════════════════════════════════════════════════════

- Se um icone nao existir, o app usa o emoji correspondente como fallback.
- Coloque os arquivos PNG nesta mesma pasta (icons/).
- Nao precisa ter todos os icones — o app funciona sem nenhum deles.
